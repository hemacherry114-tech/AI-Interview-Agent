# AI Interview Assistant

An AI-powered interview preparation platform built using FastAPI, Gemini API, JavaScript, and Render deployment.

## Features

- AI-generated interview questions
- Resume-based interview preparation
- AI answer evaluation and feedback
- Voice input support
- PDF resume upload
- Live deployed backend and frontend

## Tech Stack

- Python
- FastAPI
- Gemini API
- HTML/CSS/JavaScript
- Render
- GitHub
